# Data Directory

Sample data for testing is in `handoff/data/`:
- `handoff/data/raw/slice_10min.ndjson` - 10-minute raw data sample
- `handoff/data/processed/features_10min.parquet` - Processed features

Use this for Week 4 replay testing.

Large data files are excluded from git. Generate more data using `scripts/ws_ingest.py` and `scripts/replay.py`.
